WaterMod

Modular water storage

Illustrations and models of low fidelity proottypes

Basic summary of the whole project

For my first year design course, we had to tackle a problem happening around the world. After much research, we decided to try to solve the problem that people in Arroyo Seco, Dominican Republic faced. The main problem not that there wasn't clean water, rather that they lacked a place to store their clean water. They would often put water into dirty vessels, which would introduce contaminants into clean water pouring out of the taps.

Our group went through a iterative design cycle. We first established our design scope. We identified the main users, as well as the needs and the constraints. We built multiple prototypes and kept making improvements. In the end, we chose a modular system which would allow users to freely attach/ detach seperate water storage units with ease.

Intraweb Loading Time Logger

Measures and logs webpage loading time for monitoring purposes

Excel sheet containing data


Source Code



One of the concerns raised by people at the company was that some web-enabled forms would take too long to load. It would also vary based on the time of day that the form was opened at. It would take longer to load during midday, when our traffic was high. I decided to start a project which would log the time every hour, then visualize the data so anyone can see the trends.

I tackled the problem by first testing out bunch of methods - Jenkins and PhantomJS, for example. I switched to I then tried requests library on Python and it worked. After experimenting for a bit, I realized using requests wasn't accurate since it literally just timed how long it took to reach the website and back. I switched to PhantomJS along with Selenium so that the script would accurately measure how long it would take for an average user to load the page on their web browser.

I also used openpyxl, which would log the time it took, as well as the time script was started. The excel sheet would then be connected to Power BI for visualization.

Process raw data from excel sheets into usable data

Excel sheet containing processed data, right-most column displaying the average

I had to manually process the data that MS Society's Finance Department used for data visualization. It involved exporting an excel sheet containing hundreds of web enabled form-related data, then finding the average days it took for them to get processed. After couple months, I decided to automate the process. I used the openpyxl library to manupulate the data within the excel sheet.
